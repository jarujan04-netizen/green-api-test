# GREEN-API Test Assignment

Тестовое задание на должность Специалист технической поддержки 2 линии.

## Описание

HTML-страница для работы с методами GREEN-API — сервиса для отправки сообщений через WhatsApp.

## Реализованные методы

- getSettings — Получить настройки инстанса
- getStateInstance — Получить состояние инстанса
- sendMessage — Отправить текстовое сообщение
- sendFileByUrl — Отправить файл по URL

## Как пользоваться

1. Зайди в личный кабинет GREEN-API и создай инстанс
2. Отсканируй QR-код и подключи номер WhatsApp
3. Открой страницу: https://arzhsnm.github.io/green-api-test/
4. Введи idInstance и ApiTokenInstance из личного кабинета
5. Нажимай кнопки и смотри ответы в поле справа
