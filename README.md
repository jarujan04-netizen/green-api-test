Описание
HTML-страница для работы с методами GREEN-API — сервиса для отправки сообщений через WhatsApp.

Реализованные методы
getSettings-Получить настройки инстанса
getStateInstance-Получить состояние инстанса
sendMessage-Отправить текстовое сообщение
sendFileByUrl-Отправить файл по URL

Как пользоваться
Зайди в личный кабинет GREEN-API и создай инстанс
Отсканируй QR-код и подключи номер WhatsApp
Открой страницу: https://jarujan04-netizen.github.io/green-api-test/
Введи idInstance и ApiTokenInstance из личного кабинета
Нажимай кнопки и смотри ответы в поле справа
